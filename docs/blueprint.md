# **App Name**: CineMagic

## Core Features:

- Add Movie: Add new movie entries with details like title, description, genre, release year, and poster image.
- View All Movies: Display a list of all movies in the system with essential details and poster images.
- Movie Details: Show detailed information about a selected movie, including a larger poster image and full details.
- Edit Movie: Allow users to modify existing movie details and update the poster image if needed.
- Delete Movie: Remove movie entries from the database and delete the corresponding poster image from the server.
- Search Movies: Enable users to search for movies by title with case-insensitive and partial match support using regex.
- Poster AI Enhancement Tool: Accept a movie description as a prompt and use an AI tool to suggest enhancements to existing poster images, allowing for quick iterative refinement of posters.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to evoke a sense of cinematic mystery and sophistication.
- Background color: Very light lavender (#F0F8FF), providing a subtle contrast that keeps the focus on the content.
- Accent color: Gold (#FFD700), used sparingly for interactive elements to signify importance.
- Body font: 'PT Sans', sans-serif, used for a modern look with warmth; suits longer blocks of text well.
- Headline font: 'Playfair', serif, a high contrast font with elegance; creates a stylish contrast with body text.
- Use minimalist icons that are easy to understand and navigate, ensuring visual consistency throughout the application.
- A grid-based layout to display movie posters and information in an organized manner.
- Subtle transition animations for a smooth user experience when navigating between pages and displaying details.
